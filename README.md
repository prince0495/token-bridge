It is a Bridge to move assets between Ethereum Sepolia and Polygon Amoy Testnet.
You can move ORCA, DONK, OGGY erc20 tokens from sepolia to polygon and and also get them back to sepolia.

## Features
- Bridge Contracts (For both evm chains)
- Indexer (Index both chains to listen specific events happening on oppoiste chains)
- Client (Client side logic to let user seamlessly work with contracts and indexer with their wallets)
